## ⚡ Quick Actions

### Copy Quiz Link (2 seconds)
```
1. Click Share button (📤) on quiz card
2. Click "Copy" button
3. ✅ Done! Link copied to clipboard
```

### Share via Email (10 seconds)
```
1. Click Share button (📤)
2. Click "Email" button
3. Email client opens
4. Add recipients
5. ✅ Send!
```

### Share via WhatsApp (5 seconds)
```
1. Click Share button (📤)
2. Click "WhatsApp" button
3. WhatsApp opens
4. Select contact/group
5. ✅ Send!
```

### Generate QR Code (15 seconds)
```
1. Click Share button (📤)
2. Click "Show QR Code"
3. Wait 1-2 seconds
4. Click "Download QR Code"
5. ✅ Saved! Print and share
```

### Share Your Results (3 seconds)
```
1. Complete quiz
2. On results page, click "Share Results"
3. ✅ Mobile: Choose app to share
   ✅ Desktop: Link copied, paste anywhere
```

## 🎯 Common Scenarios

### Scenario 1: Share Quiz in Class WhatsApp Group

**Steps:**
```
1. Go to Dashboard
2. Find your quiz
3. Click Share button (📤)
4. Click "WhatsApp" button
5. WhatsApp opens in browser/app
6. Select your class group
7. Click Send
```

**Time:** ~10 seconds
**Result:** All students get quiz link instantly

---

### Scenario 2: Email Quiz to Students

**Steps:**
```
1. Go to Dashboard
2. Click Share button (📤) on quiz
3. Click "Email" button
4. Email client opens with template
5. Add student emails (or class mailing list)
6. Click Send
```

**Time:** ~30 seconds
**Result:** Professional email with quiz link sent

---

### Scenario 3: Print QR Code for Classroom

**Steps:**
```
1. Click Share button (📤)
2. Click "Show QR Code"
3. Wait for QR code to generate
4. Click "Download QR Code"
5. Open downloaded PNG
6. Print on A4 paper
7. Post in classroom
```

**Time:** ~2 minutes (+ printing)
**Result:** Students scan with phone to access quiz

---

### Scenario 4: Share High Score on Social Media

**Steps:**
```
1. Complete quiz
2. View results page
3. Click "Share Results" button
4. On mobile: Select Instagram/Twitter/etc.
   On desktop: Paste from clipboard
5. Post!
```

**Time:** ~15 seconds
**Result:** Friends see your achievement

## 💡 Tips & Tricks

### Tip 1: Multiple Sharing Methods
**Maximize reach by using all three methods:**
- Email for official notification
- WhatsApp for quick access
- QR code for in-person classes

### Tip 2: QR Code Placement
**Best places to post QR codes:**
- Classroom walls
- Notice boards
- Study halls
- Library
- Cafeteria
- Dorm common areas

### Tip 3: Customize Email Message
**After clicking Email:**
- Modify the subject line
- Add personal greeting
- Set deadline
- Add instructions
- Include study tips

### Tip 4: WhatsApp Group Rules
**Before sharing in groups:**
- Announce the quiz first
- Explain the purpose
- Set expectations
- Provide deadline
- Offer to answer questions

### Tip 5: Track Engagement
**After sharing:**
- Check Dashboard for "Attempts" count
- Monitor real-time participation
- Follow up with non-participants
- View results in Student Results page

## 🚨 Troubleshooting Quick Fixes

### "Copy button not working"
**Fix:** Click inside the input field, select all (Ctrl+A), copy (Ctrl+C)

### "Email doesn't open"
**Fix:** Manually copy the link and paste in your email client

### "WhatsApp shows error"
**Fix:** Copy link and share manually in WhatsApp

### "QR code doesn't generate"
**Fix:** Refresh page and try again. If still fails, use copy link method

### "Students can't access quiz"
**Fix:** Make sure you shared the complete URL. Test it yourself first.
