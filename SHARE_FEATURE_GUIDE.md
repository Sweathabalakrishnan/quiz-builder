# 🔗 Share Feature Guide

## Overview
Your Quiz Builder now includes comprehensive sharing functionality to easily distribute quizzes and share results!

---

## 📤 Share Quiz Feature

### Location
- **Dashboard**: Click the **Share** button (📤 icon) on any quiz card

### Features

#### 1. **Copy Link** ✂️
- **One-click copy** of the quiz URL to clipboard
- Direct link format: `https://yourdomain.com/quiz/quiz-id`
- Share via any platform (Email, WhatsApp, Slack, Discord, etc.)

#### 2. **Email Sharing** 📧
- Pre-formatted email template
- Opens default email client
- Includes:
  - Professional subject line
  - Quiz title
  - Direct quiz link
  - Friendly message

#### 3. **WhatsApp Sharing** 💬
- One-click WhatsApp share
- Works on both mobile and desktop
- Pre-formatted message with emojis
- Direct quiz link included

#### 4. **QR Code** 📱
- **Generate QR Code** for mobile scanning
- Students can scan with phone camera
- **Download QR Code** as PNG image
- Perfect for:
  - Printing and posting in classrooms
  - Adding to presentations
  - Sharing on social media
  - Including in study materials

---

## 🎯 Share Results Feature

### Location
- **Quiz Results Page**: Click **Share Results** button after completing a quiz

### Features

#### Mobile Native Share (on supported devices)
- Uses device's native share menu
- Share to any app installed on the device
- Works on:
  - iOS Safari
  - Android Chrome
  - Modern mobile browsers

#### Desktop/Fallback
- Automatically copies results to clipboard
- Share text includes:
  - 🎓 Quiz title
  - 👤 Student name
  - 📊 Score and percentage
  - 🏆 Grade
  - ✅ Pass/Fail status
  - 🔗 Link to take the quiz

## 🔧 Troubleshooting

### "Share button not working"
- Check browser clipboard permissions
- Try copying link manually
- Use alternative share methods

### "QR Code not generating"
- Check internet connection
- Refresh the page
- Try again in a few seconds

### "WhatsApp share opens wrong app"
- Make sure WhatsApp is installed
- Update WhatsApp to latest version
- Try using web.whatsapp.com

### "Students can't access quiz"
- Verify quiz link is complete
- Check if app is deployed online
- Ensure students have internet access
